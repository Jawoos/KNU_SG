# 경북대 수강신청 잔여 자리 찾기(아직 작업중!!!)

경북대 수강신청시 내가 수강하고 하는 과목의 잔여자리와 그와 같은 과목코드를 가진 과목들의 잔여자리를 한번에 확인할 수 있도록 만든 프로그램이다.
같은 과목코드를 가진 강의들은 내가 검색한 과목과의 시간을 기준으로 같은 시간인지 다른 시간인지 구분된다.
